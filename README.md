### 维修平台
主要实现以下目标：
1.	设备维修：实现设备故障报修、派工、维修、跟踪全流程化管理。
### 说明
由于本项目是采用H5混合应用开发，所以请使用HBuilder或者HBuilderX打开，使用真机运行或者在Android模拟器中进行运行。
### 技术
MUI、Vue2.x、HTML5+
